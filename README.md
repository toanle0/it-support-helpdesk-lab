# IT Support Help Desk Lab

## Overview

This repository demonstrates core IT support workflows including troubleshooting, incident documentation, system diagnostics, and knowledge base creation. The goal of this project is to simulate real-world IT support scenarios such as resolving user issues, documenting service tickets, and providing technical solutions.

## Skills Demonstrated

* Level 1 / Level 2 troubleshooting
* Technical documentation
* Incident tracking
* System diagnostics
* End-user support
* Knowledge base creation
* PowerShell troubleshooting scripts

## Tools

* Windows 11
* PowerShell
* Virtual Machines
* Remote troubleshooting concepts

## Repository Structure

tickets/
Simulated help desk incident tickets.

knowledge-base/
Technical documentation and troubleshooting guides.

scripts/
PowerShell scripts used for system diagnostics.

lab-setup/
Explanation of the simulated IT support environment.

To run the IT support diagnostic demo:

powershell

This script simulates common IT support troubleshooting tasks including system diagnostics, network connectivity testing, disk usage analysis, and process monitoring.
