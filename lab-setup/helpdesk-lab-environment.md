IT Help Desk Lab Environment

Purpose
This lab simulates a basic IT support environment used to troubleshoot user issues and document solutions.

Environment

* Windows 11 virtual machine
* PowerShell troubleshooting tools
* Simulated help desk tickets
* Knowledge base documentation

Testing Scenarios

* password reset
* printer offline troubleshooting
* system performance analysis
* Outlook connectivity issues

Outcome
The lab demonstrates practical IT support skills including troubleshooting, documentation, and system diagnostics.
