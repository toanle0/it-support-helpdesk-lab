KB-002

Title
Fix Printer Showing Offline in Windows

Symptoms

* Printer appears offline
* Print jobs stuck in queue
* Unable to print from applications

Troubleshooting Steps

1. Verify printer power and network connection
2. Restart the printer
3. Restart the Print Spooler service
4. Clear the print queue
5. Reinstall printer drivers if necessary

Resolution

Restarting the Print Spooler service and reconnecting the printer to the network restored normal printing functionality.
